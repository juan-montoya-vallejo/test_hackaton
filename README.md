## 🎯 Mission

The objective is to modernize the data stack for *MediSynth*, a legacy healthcare provider.

Participants must migrate archaic data into a
**Medallion Architecture (Bronze → Silver → Gold)** using
**AI-Driven Engineering** frameworks (**BMAD**) and **LLMs**.

---

## 🧭 Rules of Engagement

1. **AI-First Approach:** use **BMAD (AI Framework)** and **LLMs**.
2. **Medallion Standard:** final output must respect
   **Bronze**, **Silver**, and **Gold** layers.
3. **The Prompt Journal:** keep a log of **all** prompts used for evaluation.

## 📊 Evaluation Criteria

| Category | Weight | Validation / Logic |
|---|---:|---|
| **Data Fidelity** | **40%** | Automated SQL check against the "Golden Record". |
| **AI/BMAD Mastery** | **30%** | Review of Prompt Journal depth and automation level. |
| **Engineering** | **20%** | Review of DQ tests and *AI-generated Lineage*. |
| **Velocity** | **10%** | Delivery time vs. manual effort estimates. |
