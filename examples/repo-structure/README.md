# Repository Structure Example

This folder includes an example of how you can organize your repository to implement
the **AI-Accelerated Migration Challenge** using a
**Medallion (Bronze → Silver → Gold)** approach.

It is not a requirement; it is a structure guide to give you a starting point.

You can implement it with **1 or more Python files**: from a minimal version
(`pipeline.py`) to a modular version (`bronze.py`, `silver.py`, `gold.py`, etc.).

Recommended practices:
- **TDD (Test-Driven Development):** start with tests, then implement.
- **OOP:** organize logic in classes/modules to keep code clear and scalable.

If you use Python, this guide supports **1 or more `.py` files per layer** and encourages
**TDD** and **OOP** to maintain quality and design clarity.
